## 📁 Assets

Esta pasta é destinada aos recursos visuais que documentam o funcionamento e a arquitetura do projeto:
* **Diagramas de Arquitetura:** Representação do fluxo de dados, mostrando como o usuário interage com o agente e como o agente consulta os manuais e logs na Base de Conhecimento.
* **Screenshots da Aplicação:** Capturas de tela demonstrando o assistente na prática (ex: interpretando um alerta de SIEM ou explicando um conceito de rede).
* **Imagens para o README:** Avatar do assistente e banners do projeto.
